# 📦 Gestión de Inventario By Sofii

¡Bienvenido/a al repositorio del proyecto **Gestión de Inventario**! 
Esta es una aplicación web interactiva diseñada para administrar y llevar el control de inventarios de manera sencilla y eficiente.

🌐 **Demo del proyecto en vivo:** [Visitar la aplicación](https://gestion-inventario-by-sofii.vercel.app)

---

## 🚀 Tecnologías Utilizadas

Este proyecto fue desarrollado utilizando tecnologías web estándar (Vanilla web), lo que garantiza un rendimiento rápido y cero dependencias complejas:

* **HTML5** (17.9%) - Estructuración y semántica de la interfaz.
* **CSS3** (27.6%) - Estilos, diseño responsive y experiencia visual.
* **JavaScript** (54.5%) - Lógica de negocio, interactividad y manipulación del DOM.
* **JSON** - Almacenamiento y estructura de los datos iniciales (`data.json`).

---

## 📁 Estructura del Proyecto

El código está organizado de manera modular para mantener las buenas prácticas y facilitar la escalabilidad:

```text
📂 Gestion-inventario-By-Sofii-
├── 📁 components/    # Componentes reutilizables de la interfaz (navbar, modales, etc.)
├── 📁 css/           # Hojas de estilo generales y específicas
├── 📁 images/        # Recursos gráficos y multimedia
├── 📁 js/            # Lógica principal de JavaScript y controladores
